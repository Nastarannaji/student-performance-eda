# Student Performance EDA Project

## 📊 Overview
This project analyzes factors influencing student exam performance using Python.

## 📁 Dataset
- Student exam scores dataset (math, reading, writing)

## 🧠 Goal
To explore what factors (gender, lunch, parental education, test prep) influence student performance.

## 🛠 Tools
- Python
- Pandas
- Seaborn
- Matplotlib

## 📈 Output
- Data analysis
- Visualizations
- Insights about student performance
